# FIT Metronom

## Technické info

Projekt je napsán pomocí grafického toolkitu Qt a jeho knihovny QtQuick.
Použitá verze Qt byla 5.12 (aktuálně předposlední LTS).

Použité knihovny:

- QtQuick 2.0
- QtQuick.Controls 2.15
- QtQuick.Layouts 1.11
- QtQml.Models 2.12
- QtMultimedia 5.12
- Qt.labs.settings 1.0

## Návod k instalaci

1. Vejděte do zdrojové složky
2. `qmake`
3. `make`
4. Po úspěšné kompilaci by měl být v aktuální složce dostupný spustitelný soubor `Metronome`.
